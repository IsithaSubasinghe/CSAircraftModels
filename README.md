# CSair

A simple Node.js + Express site for showcasing handcrafted aircraft models and handling custom order requests.

## Project structure

- public/ - frontend assets served by the app
- src/ - source/helper files
- backend-data/ - order data, uploads, and SQLite database
- server.js - Express server entry point

## Run locally

```bash
npm install
npm start
```

The app will be available at http://localhost:3001.
