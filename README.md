

# ✈️ CS Aircraft Models

A modern full-stack web application developed for **CS Aircraft Models**, a business specializing in handcrafted wooden aircraft models. The website provides an engaging platform to showcase products, share the artist's story, and streamline customer inquiries and custom aircraft orders.

# 📖 Overview

CS Aircraft Models required a professional online presence to showcase their handcrafted wooden aircraft models while making it easier for customers to inquire about products and request custom builds.

This project was designed and developed with a focus on responsive design, clean user experience, and reliable backend functionality.

---

# ✨ Features

- Responsive modern user interface
- Aircraft model showcase
- Interactive image galleries
- Artist profile section
- Custom aircraft order request form
- Contact & inquiry system
- Secure file upload support for custom requests
- Automated email notifications
- Mobile-friendly design
- Smooth navigation and optimized user experience
